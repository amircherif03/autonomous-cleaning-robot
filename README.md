# Autonomous Cleaning Robot — PFA 2026

> Final Year Project (PFA) — INSAT Tunis | Embedded Systems & Robotics

## Overview
Autonomous cleaning robot capable of analyzing its environment using an AI agent
and dynamically selecting the optimal cleaning mode. Full mechanical design and
embedded control architecture developed from scratch.
